Part 1 — Node Degree

What Node Degree Tells You

The degree of a node is the simplest and most fundamental structural invariant. It's the starting point of WL refinement.

📐
Definition
deg(v) = number of edges incident to v. In an undirected graph, this equals the count of v's neighbors. It's the 0-iteration WL color.
📊
Degree Sequence
The sorted list of all node degrees. Must be identical in both graphs for isomorphism to be possible. A mismatch is instant proof of non-isomorphism.
⚠️
Anomalies
Nodes with unique degrees (appear in only one graph) are highlighted in red in the degree table — they're strong structural indicators.

Part 3 — Iteration Differential

Understanding Color Divergence

The "first-difference iteration" is the most critical output — the earliest point where the two graphs produce different color partitions.

ITER 0
Initial coloring by degree
Both graphs assigned colors by node degree. If degree sequences differ here, they're immediately non-isomorphic.
ITER 1
First neighborhood refinement
Each node's color updated with neighbor information. Local structure starts to differentiate nodes.
ITER 2 ⚡
Divergence detected (example)
The first iteration where the two graphs produce different color class counts or distributions. This is the definitive evidence — the graphs cannot be isomorphic.
CONV
Convergence — stable partition
Colors no longer change between iterations. The algorithm terminates and final certificates are compared.
Quick Reference

Color Coding Guide

✅
Green Signal
Check passed. Degree or WL certificate matches between both graphs.
❌
Red Signal
Check failed. Definitive structural difference found. Graphs are NOT isomorphic.
⚠️
Amber Tab
This iteration is where the graphs first diverge in color partition.
🔵
Blue Node
Node is color class 0. Graph nodes are colored by their current WL color class.
🔢
Color Count
Number of distinct color classes in the current partition. More colors = more discriminating.
⏸
Convergence
No new color classes added. WL partition is stable. Algorithm complete.

02 / Isomorphism

Graph Isomorphism
& Structural Equivalence

Two graphs are isomorphic if there exists a bijection between their vertex sets that preserves all edges. This is one of the deepest problems in combinatorics.

Definition

What is Graph Isomorphism?

Informally: two graphs are "the same" if you can relabel the nodes of one to get the other. The exact structure of connections — not the labels — is what matters.

🔗
Edge Preservation
A bijection f: V(G₁) → V(G₂) is an isomorphism if (u,v) ∈ E(G₁) ⟺ (f(u), f(v)) ∈ E(G₂) for all pairs u,v.
🧩
Structural Identity
Isomorphic graphs are structurally identical — same connectivity patterns, same cycles, same vertex degrees, same eigenvalues of their adjacency matrices.
🔢
Complexity
Graph isomorphism sits in NP but is not known to be NP-complete. It's one of very few natural problems with this property (GI-complete class).
Necessary Conditions

Quick Non-Isomorphism Checks

These invariants are necessary for isomorphism. If any fails, the graphs are definitely not isomorphic. They're fast to compute.

📊
Same n and m
Same number of nodes and edges. Different counts → immediately not isomorphic.
📈
Degree Sequence
The sorted list of node degrees must be identical. A quick O(n log n) filter.
🔄
Cycle Structure
Same number of cycles of each length. Same girth (shortest cycle). Same chromatic number.
🌳
Spectrum
Eigenvalues of the adjacency matrix must match. Cospectral graphs can still be non-isomorphic (Seidel 1957).

Limitations

When WL Can Fail

The WL test is powerful but not complete. Here are important cases where it cannot distinguish non-isomorphic graphs:

⚠️
Regular Graphs
Two k-regular graphs with the same number of vertices always have identical WL certificates after iter 0, regardless of structure. More iterations don't help.
🔁
CFI Graphs (Cai-Fürer-Immerman)
Specifically constructed graph families that defeat 1-WL. Even higher k-WL can be defeated by appropriate CFI constructions.
🧩
Strongly Regular Graphs
Many non-isomorphic strongly regular graphs (e.g., Petersen vs. others with the same parameters) have identical 1-WL certificates.
📐
Completeness Gap
For any fixed k, there exist non-isomorphic graphs that k-WL cannot distinguish. Only the full GI algorithm is complete.

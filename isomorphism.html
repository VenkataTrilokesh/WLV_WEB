Graph Isomorphism

Two graphs are isomorphic if they are structurally identical — the same connections, just with the nodes drawn or labelled differently. Think of it as two different maps of the same city.

What does "isomorphic" actually mean?

Imagine two social networks. Even if the people have different names, if every friendship pairing is identical, the two networks are isomorphic — they are the same underlying structure in disguise.

This matters in chemistry (two molecules with the same structure behave the same way), network analysis, and pattern matching.

How hard is it to check?

Checking whether two graphs are isomorphic is famously tricky — it sits in a grey zone between "easy" and "hard" problems in computer science. No one has proven it's easy, and no one has proven it's hard either.

Fast
Simple graph types

Certain shapes — like trees or road-map-style graphs — can be checked quickly and efficiently.

Mid
General graphs

A 2015 breakthrough produced a near-fast algorithm. Practical tools like nauty handle most real-world graphs efficiently.

Hard
Subgraph search

Asking "does graph G contain graph H somewhere inside it?" is provably hard — this version of the problem is NP-complete.

Quick checks before a full test

You can often rule out isomorphism cheaply, without running a full algorithm. If any of these don't match, the graphs are definitely different:

  • Same number of nodes and edges
  • Same degree sequence — each node's number of connections, sorted
  • Same number of connected components

A mismatch in any of these is a guaranteed proof the graphs are not isomorphic.

Where the WL test fits in

The Weisfeiler–Lehman (WL) test is a fast way to compare graphs, but it's not perfect:

  • If WL says "different" — the graphs definitely are. No further checking needed.
  • If WL says "same" — they probably are, but it's not guaranteed. Some rare graph pairs fool the test.

Using a higher value of k makes the test stricter and catches more cases. For a guaranteed answer, exact solvers like nauty can be used.
